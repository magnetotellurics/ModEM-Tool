edi2list is a fortran command line tool for reading a collection of EDI files and outputting in list format of ModEM. Use 'make' to compile and test it. There is an example of using edi2list in 'test' folder. edi2list first read 'runfile' - configuration file to get all the parameters. This fortran code is also an example that shows how to read free-style(keywords, comments, et al) config file by using fortran.
